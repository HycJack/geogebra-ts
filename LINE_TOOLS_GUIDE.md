# 线工具使用说明

## 概述

线工具提供了四种几何线的创建功能：
- **垂线** - 通过点创建与直线垂直的线
- **平行线** - 通过点创建与直线平行的线
- **中垂线** - 创建两点连线的中垂线
- **角平分线** - 创建角的平分线

## 访问方式

### 方式一：在主界面使用
1. 打开浏览器访问 http://localhost:5174/
2. 在工具栏中找到"线工具"按钮（带有下拉箭头的图标）
3. 点击"线工具"按钮，会显示下拉菜单
4. 从下拉菜单中选择你想要的线工具

### 方式二：使用演示页面
1. 在主页面点击右上角的"线工具演示"按钮
2. 进入专门的线工具演示页面
3. 在左侧面板选择线工具模式
4. 在右侧画布上点击创建线

## 各工具详细说明

### 1. 垂线工具

#### 功能
创建通过指定点且与给定直线垂直的直线。

#### 操作方式

**方式一：先选择直线，再选择点**
1. 点击一条已有的直线（直线会被高亮显示）
2. 点击一个点（可以是新点或已有点）
3. 系统自动创建通过该点且垂直于选定直线的线

**方式二：先选择点，再选择点**
1. 点击第一个点（垂线将通过的点）
2. 点击第二个点（用于确定垂线方向的点）
3. 系统创建通过第一个点且垂直于两点连线的线

#### 使用场景
- 创建三角形的垂线
- 构造点到直线的垂线
- 绘制垂直辅助线

#### 示例
```
步骤1: 点击直线 AB
步骤2: 点击点 C
结果: 创建通过点 C 且垂直于直线 AB 的线
```

### 2. 平行线工具

#### 功能
创建通过指定点且与给定直线平行的直线。

#### 操作方式

**方式一：先选择直线，再选择点**
1. 点击一条已有的直线（直线会被高亮显示）
2. 点击一个点（可以是新点或已有点）
3. 系统自动创建通过该点且平行于选定直线的线

**方式二：先选择点，再选择点**
1. 点击第一个点（平行线将通过的点）
2. 点击第二个点（用于确定平行线方向的点）
3. 系统创建通过第一个点且平行于两点连线的线

#### 使用场景
- 创建平行四边形
- 绘制平行辅助线
- 构造平行线相关几何图形

#### 示例
```
步骤1: 点击直线 AB
步骤2: 点击点 C
结果: 创建通过点 C 且平行于直线 AB 的线
```

### 3. 中垂线工具

#### 功能
创建两点连线的中垂线（垂直平分线）。

#### 操作方式
1. 点击第一个点（线段的第一个端点）
2. 点击第二个点（线段的第二个端点）
3. 系统自动创建两点连线的中垂线

#### 使用场景
- 构造三角形的外心
- 绘制线段的垂直平分线
- 求解几何问题中的中垂线

#### 示例
```
步骤1: 点击点 A
步骤2: 点击点 B
结果: 创建线段 AB 的中垂线
```

### 4. 角平分线工具

#### 功能
创建角的平分线。

#### 操作方式
1. 点击角的顶点
2. 点击角的第一条边上的点
3. 点击角的第二条边上的点
4. 系统自动创建该角的平分线

#### 使用场景
- 构造三角形的内心
- 绘制角的平分线
- 求解几何问题中的角平分线

#### 示例
```
步骤1: 点击顶点 O
步骤2: 点击边 OA 上的点 A
步骤3: 点击边 OB 上的点 B
结果: 创建角 AOB 的平分线
```

## 界面说明

### 工具面板
- **线工具选择区**：四个工具按钮，点击切换工具模式
- **操作步骤提示**：显示当前步骤和进度指示器
- **操作提示**：详细的操作说明
- **操作方式说明**：展示可用的操作方式
- **键盘快捷键**：可用的快捷键列表
- **取消按钮**：取消当前操作

### 视觉反馈
- **步骤指示器**：蓝色圆点表示已完成的步骤，灰色圆点表示未完成的步骤
- **高亮显示**：可选择的直线会被高亮显示
- **预览效果**：在操作过程中会显示预览效果
- **完成提示**：操作完成后显示绿色成功提示

## 键盘快捷键

- **Esc** - 取消当前操作
- **Enter** - 确认当前操作（垂线和平行线工具）

## 注意事项

1. **点的选择**
   - 可以点击画布上的任意位置创建新点
   - 也可以点击已有的点
   - 系统会自动吸附到附近的点

2. **直线的选择**
   - 垂线和平行线工具支持选择已有直线
   - 鼠标悬停在直线上时，直线会高亮显示
   - 点击直线即可选中

3. **操作取消**
   - 在任何时候按 Esc 键可以取消当前操作
   - 点击"取消当前操作"按钮也可以取消
   - 取消后可以重新开始操作

4. **错误处理**
   - 如果选择的点重合，系统会提示错误
   - 如果选择的直线无效，系统会提示错误
   - 请按照提示重新选择

## 实用技巧

### 快速创建多条线
1. 完成一条线的创建后，可以立即开始创建下一条线
2. 不需要重新选择工具模式
3. 系统会自动重置步骤

### 使用已有几何对象
1. 先创建基本的点和直线
2. 然后使用线工具创建相关的几何线
3. 这样可以构建复杂的几何图形

### 组合使用多个工具
1. 可以组合使用垂线、平行线、中垂线等工具
2. 构建复杂的几何图形和定理证明
3. 例如：使用垂线和中垂线构造三角形的外心

## 常见问题

**Q: 如何创建通过特定点的垂线？**
A: 先点击要垂直的直线，然后点击要通过的点。

**Q: 中垂线工具和垂线工具有什么区别？**
A: 中垂线工具创建的是两点连线的中垂线，垂线工具创建的是通过一点且垂直于给定直线的线。

**Q: 角平分线工具需要点击几个点？**
A: 需要依次点击三个点：顶点、第一条边上的点、第二条边上的点。

**Q: 如何取消当前操作？**
A: 按 Esc 键或点击"取消当前操作"按钮。

**Q: 可以创建多条平行的线吗？**
A: 可以，完成一条平行线后，可以继续创建其他平行线。

## 技术实现

### 算法
- **垂线算法**：使用向量叉积计算垂线方程
- **平行线算法**：使用相同斜率计算平行线方程
- **中垂线算法**：计算两点中点，然后创建垂线
- **角平分线算法**：使用角平分线定理计算方程
